Redeployment Guide

If you are using the static file mode, follow these steps to upload your changes permanently to the internet:

  1. Click "Download Updated Config" below to download your new portfolio_data.json file.
  2. Replace the old portfolio_data.json inside your local portfolio folder with this downloaded file.
  3. Run the firebase deployment script in your terminal to sync files to your hosting server.
